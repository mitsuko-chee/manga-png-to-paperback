# manga-png-to-paperback

AI漫画のPNG画像をKDPペーパーバック用PDFに変換するツールです。

---

## 受け取った方へ：セットアップ方法

詳しい手順は [INSTALL.md](INSTALL.md) を見てください。

ターミナルで以下を実行するとスキルが自動でインストールされます。

```bash
curl -fsSL https://raw.githubusercontent.com/mitsuko-chee/manga-png-to-paperback/main/install.sh | bash
```

---

## 使い方

セットアップ後、Claude Codeで以下のように呼び出せます。

```
/manga-png-to-paperback
```

---

## 実行中に確認されること

起動後、以下の順番で質問されます。答えに応じてPDFが生成されます。

1. PNG画像が入ったフォルダのパス
2. 判型（A5 / B6）
3. 裁ち落とし（あり / なし）
4. ノンブル（ページ番号）を入れますか（あり / なし）
5. ノンブルをスキップするページはありますか（扉など）→ ありの場合、先頭から何ページスキップするか数字で入力
6. 余白の追加確認

なお、暗いページのノンブルは自動で白抜きになります。

設定を変更したい場合はClaude Codeに日本語で指示してください。

---

## PNG画像の準備

- 解像度：300dpi推奨（A5なら1748×2480px以上）
- ファイル形式：PNG
- ファイル名：001.png、002.png のように連番にしてください

---

## 注意事項

- 最終的なKDP入稿前には、必ずKDPプレビューで確認してください
- KDP入稿画面のサイズ指定：A5は148×210mm、B6は128×182mm
