CanvaなどでエクスポートしたPNGファイルを、Kindle KDP入稿用のPDFに変換する。

## 実行方法

以下のコマンドを実行する。PNGフォルダのパスはユーザーに確認する。

```bash
python3 ~/my-workspace/manga_pdf_maker.py
```

スクリプトが対話式で以下を順番に質問するので、ユーザーの回答をそのまま入力する：

1. PNG画像が入ったフォルダのパス
2. 判型（A5 / B6）
3. 裁ち落とし（あり / なし）
4. ノンブル（あり / なし）
5. ノンブル開始ページ（ありの場合）― ファイル名の数字部分を入力（例: 001）
6. スキップするページ（ありの場合）― ノンブルを表示しないページをドット区切りで入力（例: 001.024.059）。不要ならEnter
7. 白抜きにしたい特定のページ（ありの場合）― ファイル名をドット区切りで入力（例: 005.010）。不要ならEnter
8. 余白の追加（y / n）― 画像にすでに余白がある場合は n、余白がない場合は y

## 注意事項

- `manga_pdf_maker.py` が見つからない場合は `install.sh` を再実行するよう伝える
- PNGファイルは 001.png、002.png のように連番にする必要がある
- スキップページ・白抜きページはファイル名（拡張子なし）をドット区切りで指定する
