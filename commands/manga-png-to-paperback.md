CanvaなどでエクスポートしたPNGファイルを、Kindle KDP入稿用のPDFに変換する。

## 実行方法

すべての質問を **AskUserQuestion（選択式ダイアログ）** で行う。テキストで聞かない。

### ステップ1：フォルダパスを確認する
AskUserQuestion でフォルダパスを聞く。

### ステップ2：基本設定をまとめて聞く
以下の4つを1回のAskUserQuestionで聞く：
- 判型（A5 / B6）
- 裁ち落とし（あり / なし）
- ノンブル（あり / なし）
- 余白の追加（y=余白なし画像 / n=余白あり画像）

**「裁ち落とし：あり」を選んだ場合、以下の警告をテキストで表示する：**
> ⚠️ 注意：「裁ち落としあり」は塗り足し（3mm）付きのテンプレートで作った画像専用です。

### ステップ3：ノンブル詳細（ノンブルありの場合のみ）
以下の3つを1回のAskUserQuestionで聞く：
- ノンブル開始ページ（ファイル名の数字部分、例: 001）
- スキップするページ（ドット区切り、例: 001.024.059。なければ「なし」）
- 強制的に白抜きにしたい特定ページ（ドット区切り、例: 005.010。なければ「なし」）

### ステップ4：スクリプトを実行する
集めた回答を使って以下を実行する：

```bash
python3 ~/my-workspace/manga_pdf_maker.py
```

## 注意事項

- `manga_pdf_maker.py` が見つからない場合は `install.sh` を再実行するよう伝える
- PNGファイルは 001.png、002.png のように連番にする必要がある
- スキップページ・白抜きページはファイル名（拡張子なし）をドット区切りで指定する
- 暗いページのノンブルは自動で白抜きになる

## フォルダパスがわからない場合

**Mac:** Finderでフォルダを右クリックし、**Optionキーを押しながら**「パス名をコピー」を選ぶ。

**Windows:** エクスプローラーでフォルダを**Shiftキーを押しながら右クリック**し、「パスのコピー」を選ぶ。

コピーしたパスをチャットに貼り付けてもらう。
